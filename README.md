# Bankist App

A simple banking app built with JavaScript to simulate basic account operations.

## Live Demo
https://bankist-app-mihailfrunze.netlify.app/

## GitHub
https://github.com/mihail-frunze/bankist-app

## How it works

- Users log in with a username and PIN
- The app shows transaction history
- You can transfer money to another user
- You can request a loan
- You can close the account
- The balance updates automatically
- Transactions can be sorted
